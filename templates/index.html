Forgery Detection Model

DenseNet121Keras .kerasELA

A DenseNet121 trained on Error Level Analysis representations of CASIA 2.0. ELA highlights JPEG compression inconsistencies left behind by image editing. The model was trained with MirroredStrategy on dual T4 GPUs.

ArchitectureDenseNet121 + FC(1024) + Softmax
Input Size128 × 128 × 3 (ELA)
PreprocessingError Level Analysis (q=90)
Training DatasetCASIA 2.0
Output ClassesREAL · TAMPERED
ELA Sweepq = 95, 90, 85, 80, 75, 70

AI Classification Model

ViTPyTorch .pthTransformer

A Vision Transformer (ViT-Base/16) fine-tuned to classify images as real or AI-generated. Patch-based attention detects global structural inconsistencies common in generative models.

ArchitectureViT-Base/Patch16-224
Input Size224 × 224 × 3
Training DatasetCIFAKE
Output ClassesREAL · AI GENERATED
